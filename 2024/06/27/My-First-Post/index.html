使用 Hexo 项目并在 GitHub 上部署指南

Hexo 简介

Hexo 是一个快速、简洁且高效的博客框架，基于 Node.js 开发，适用于个人和团队博客的搭建。

步骤一：安装 Hexo

  1. 安装 Node.js 和 npm

    • 访问 Node.js 官网 下载并安装最新版本的 Node.js。
    • npm 会随 Node.js 一同安装。

  2. 安装 Hexo

    npm install -g hexo-cli
初始化 Hexo 项目

复制代码
hexo init myblog
cd myblog
npm install

步骤二：配置 Git 与 GitHub

-在 GitHub 上创建仓库

-登录 GitHub，创建一个新的空仓库，例如 username.github.io（其中 username 是你的 GitHub 用户名）。
-配置 Git

-初始化 Git 仓库并关联远程仓库：

bash
复制代码
git init
git remote add origin https://github.com/username/username.github.io.git
编写文章与提交更改

使用 Hexo 创建新文章：
bash
复制代码
hexo new “Hello World”
编辑文章并保存。
生成静态文件

bash
复制代码
hexo generate
部署到 GitHub Pages

安装 Hexo 部署插件：
bash
复制代码
npm install hexo-deployer-git –save
配置 _config.yml 文件：
yaml
复制代码
deploy:
type: git
repo: https://github.com/username/username.github.io.git
branch: master
部署到 GitHub：
bash
复制代码
hexo deploy
步骤三：Git 基本操作
查看文件状态

bash
复制代码
git status
添加文件到暂存区

bash
复制代码
git add .
提交更改到本地仓库

bash
复制代码
git commit -m “提交说明”
推送本地更改到远程仓库

bash
复制代码
git push origin master
步骤四：Hexo 常用命令
启动本地服务器

bash
复制代码
hexo server
清除缓存文件

bash
复制代码
hexo clean
生成静态文件

bash
复制代码
hexo generate
步骤五：Hexo 主题与配置
安装 Hexo 主题

在 GitHub 上找到喜欢的 Hexo 主题，例如：
bash
复制代码
git clone https://github.com/theme-author/theme-name.git themes/theme-name
配置主题

修改 _config.yml 文件，指定使用的主题：
yaml
复制代码
theme: theme-name
自定义配置

根据主题文档修改 _config.yml 文件，配置个性化内容。
